# 🟩 Vue.js Bootcamp

A structured, lesson-by-lesson **Vue.js learning path** designed to get from *getting started* to building real, maintainable front-end applications.  
This repository is organized like a bootcamp: each module focuses on a core Vue concept and includes practical notes, exercises, and project-focused guidance.

---

## ✨ Features

- Clear, **modular structure** (each folder is a lesson / topic)
- Covers both **fundamentals** and **application architecture**
- Includes **debugging notes** and practical developer tips
- Great for **self-study**, **revision**, or as a **teaching companion**

## 📝 Overview

This repo contains a curated set of notes and learning materials created during the study of Vue.js front-end development for the Computer Science course "Web and Software Architecture" from Sapienza University of Rome.

It's possible to progress through topics such as:
- Vue fundamentals and core concepts
- Building UIs with reusable components
- State management with Vuex
- Routing with Vue Router
- Integrating APIs with Axios
- Consolidating everything into a Vue project

---

## 🧭 Modules

> Folder names are numbered to follow a suggested learning order.

### ✅ [01] GETTING STARTED
Environment setup, first steps, and essential tooling to begin working effectively with Vue.

### 🧠 [02] CORE CONCEPTS
The foundations of Vue development (reactivity, directives, computed properties, lifecycle, etc.).

### 🧩 [03] BUILDING USER INTERFACES
Patterns and techniques for building clean, scalable UI layers.

### 🧱 [04] COMPONENTS
Reusable component design, props/events, component communication, composition patterns, and structure.

### 🗃 [05] VUEX
State management fundamentals: store structure, mutations/actions/getters, and app-scale patterns.

### 🧭 [06] VUE ROUTER
Routing fundamentals: route configuration, navigation, nested routes, and guards.

### 🌐 [07] WORKING WITH APIS - AXIOS
Calling REST APIs, handling async data, errors, loading states, and integration best practices.

### 🚀 [09] VUE PROJECT
A project-focused area to consolidate the modules into a cohesive application approach.

## 🚀 Getting Started

### Prerequisites
- **Node.js** (recommended: LTS)
- **npm** or **yarn**
- Git

### Clone the repository
```bash
git clone https://github.com/GCRA101/Vue.js-Bootcamp.git
cd Vue.js-Bootcamp
```

> Note: This repository is organized primarily as a learning resource.  
> If you want to run a specific project inside `[09] VUE PROJECT`, open that folder and follow its setup steps (package.json / scripts) if present.

---

## 🛠 Recommended Workflow

1. Start with **[01] GETTING STARTED** and move forward in numeric order.
2. Take notes directly inside each module folder (or your own notes app).
3. Practice by building a mini-feature per module (example: a component library, a router-based layout, an API-driven list view).
4. Consolidate everything in **[09] VUE PROJECT**.
